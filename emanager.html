Huahin EManager

Huahin EManager REST API is replaced with the Amazonn Elastic MapReduce(EMR) API in products for managing EMR Job Flows. It also has its own queue management and a different work queue of EMR. EManager can limit the number of clusters that can be run simultaneously. Also, EManager is optimized to reduce the cost and start the instance. The main function is as follows:

  • Get job flow list.
  • Get describe job flow.
  • Kill step for step name.
  • Get queue list.
  • Get describe queue.
  • Register job.(Hive, Pig, Streaming and Custom JAR)

Kill of step has been achieved by using the Huahin Manager. Refer to the Huahin Manager for more information.
